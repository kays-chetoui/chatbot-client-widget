Introduction

What is the Chatbot Widget?

Our chatbot widget is a lightweight and high-performance solution that allows you to easily add a conversational assistant to your website. It's designed to be:

  • Simple to integrate - add just a few lines of code to your site
  • Highly customizable - adapt the appearance to match your brand
  • Performant - minimal loading time for your visitors
  • Responsive - works perfectly on mobile, tablet, and desktop

Quick Installation

Integration in 2 steps

1

Include the necessary files

Add these lines in the <head> section of your HTML page:

<!-- Tailwind CSS -->
<script src="./dist-widget/tailwindcss.js"></script>
<!-- Chatbot CSS -->
<link rel="stylesheet" href="./dist-widget/chatbot-widget.css" />
2

Add the script with configuration

Insert this script just before the closing </body> tag:

<!-- Chatbot Script -->
<script 
  src="./dist-widget/chatbot-widget.js" 
  data-chatbot-widget 
  data-chatbot-config='{
    "title": "My Assistant", 
    "welcomeMessage": "Hello! How can I help you?", 
    "position": "bottom-right", 
    "avatarBot": "./src/assets/avatars/bot.svg", 
    "avatarUser": "./src/assets/avatars/user.svg",
    "themeMode": "light", 
    "enableLightDarkToggle": true,
    "theme": {
      "primary": "#2563EB", 
      "messageBg": {
        "user": "#2563EB", 
        "bot": "#DBEAFE"
      }, 
      "messageText": {
        "user": "#FFFFFF", 
        "bot": "#1F2937"
      }
    },
    "accessibility": {
      "fontSize": "medium",
      "reducedMotion": false,
      "highContrast": false,
      "keyboardShortcuts": true
    }
  }'
></script>

Important: Make sure to adjust the file paths (path to dist-widget) according to your project structure.

Customization Options

Customize the appearance and behavior of the chatbot by modifying the options in the data-chatbot-config attribute:

Appearance

  • title
    Title in the header bar

    Ex: "Customer Service", "Virtual Assistant"

  • theme
    Color customization

    Object with several properties:

    • primary: main color (buttons, headers)
    • messageBg: background colors for message bubbles (user and bot)
    • messageText: text colors for messages (user and bot)
  • themeMode
    Light or dark mode

    Available options: light, dark, system

  • enableLightDarkToggle
    Allow users to switch themes

    Boolean value: true or false

  • avatarBot
    Bot avatar image URL

    Path to image file for bot avatar

  • avatarUser
    User avatar image URL

    Path to image file for user avatar

Behavior & Accessibility

  • welcomeMessage
    Initial welcome message

    First message displayed when the chat opens

  • position
    Position on the screen

    Available options:

    • bottom-right (default)
    • bottom-left
    • top-right
    • top-left
  • accessibility
    Accessibility settings

    Object with 4 properties:

    • fontSize: text size (small, medium, large, xlarge)
    • reducedMotion: reduce animations (true/false)
    • highContrast: increase contrast (true/false)
    • keyboardShortcuts: enable keyboard navigation (true/false)

Advanced Installation (JavaScript)

Programmatic Initialization

For developers who prefer to initialize the chatbot via JavaScript, follow these steps:

1

Import the required files

Include the CSS and JavaScript script in your page (without the data-chatbot-widget attribute):

<!-- In the head section -->
<link rel="stylesheet" href="./dist-widget/chatbot-widget.css" />

<!-- Before the body closing tag -->
<script src="./dist-widget/chatbot-widget.js"></script>
2

Initialize with JavaScript

Call the init() method with your configuration options:

<script>  // Initialize the chatbot when the page is loaded
  document.addEventListener('DOMContentLoaded', function() {
    window.ChatbotWidget.init({
      title: "Technical Support",
      welcomeMessage: "Hello! How can I assist you today?",
      position: "bottom-right",
      avatarBot: "./src/assets/avatars/bot.svg",
      avatarUser: "./src/assets/avatars/user.svg",
      themeMode: "light",
      enableLightDarkToggle: true,
      theme: {
        primary: "#4F46E5",
        messageBg: {
          user: "#4F46E5",
          bot: "#EEF2FF"
        },
        messageText: {
          user: "#FFFFFF",
          bot: "#1F2937"
        }
      },
      accessibility: {
        fontSize: "medium",
        reducedMotion: false,
        highContrast: false,
        keyboardShortcuts: true
      }
    });
  });
</script>

Advanced JavaScript Features

Code initialization also allows you to use these additional methods:

Open/Close the chatbot

// Open the chatbot
window.ChatbotWidget.open();

// Close the chatbot
window.ChatbotWidget.close();

// Toggle the chatbot open/closed
window.ChatbotWidget.toggle();

Send messages

// Send a simple text system message
window.ChatbotWidget.sendSystemMessage(
  "Here's some important information!"
);

// Send a rich content system message
window.ChatbotWidget.sendSystemMessage([
  { type: 'text', value: 'Here is some formatted content:' },
  { type: 'code', value: 'console.log("Hello world!");', language: 'javascript' },
  { type: 'link', value: 'https://example.com', label: 'Visit our site' }
]);

Initialization tracking

// Check if widget is initialized
if (window.ChatbotWidget.isInitialized()) {
  // Do something with the widget
}

// Or use the callback approach
window.ChatbotWidget.onInitialized(function() {
  // Safe to use widget methods here
  window.ChatbotWidget.open();
});

// Or use async/await with the promise
await window.ChatbotWidget.waitForInitialization();
window.ChatbotWidget.open();

Theme & Accessibility Control

// Change theme mode dynamically
window.ChatbotWidget.setThemeMode('dark');  // 'light', 'dark' or 'system'

// Update accessibility options
window.ChatbotWidget.updateAccessibilityOptions({
  fontSize: 'large',
  reducedMotion: true,
  highContrast: true,
  keyboardShortcuts: true
});

Event Listeners

// Listen for user message events
document.addEventListener('chatbot-user-message', function(event) {
  console.log('User sent message:', event.detail);
  
  // You can send an automated response
  setTimeout(() => {
    window.ChatbotWidget.sendSystemMessage('Processing your request...');
  }, 500);
});

// Listen for chatbot open/close events
document.addEventListener('chatbot-opened', function() {
  console.log('Chatbot was opened');
});

document.addEventListener('chatbot-closed', function() {
  console.log('Chatbot was closed');
});

Rich Messages & Accessibility

Rich Content Messages

The chatbot widget supports various types of rich content messages, allowing you to create interactive and engaging conversations:

// Send a message with multiple content types
window.ChatbotWidget.sendSystemMessage([
  { type: 'text', value: 'Here are some options:' },
  { type: 'link', value: 'https://example.com', label: 'Visit our website' },
  { type: 'image', value: 'https://example.com/image.jpg', alt: 'Product image', caption: 'Our latest product' },
  { type: 'code', value: 'npm install my-package', language: 'bash' },
  { type: 'html', value: '<b>Important notice</b>' },
  { 
    type: 'card', 
    value: {
      title: 'Product Card',
      image: 'product.jpg',
      price: 99.99,
      rating: 4.8
    },
    actionLink: 'https://store.example.com/buy' 
  }
]);
Content Type Description
text Simple text message
link Clickable URL with optional label
image Image with caption and alt text
code Code snippet with language highlighting
html Formatted HTML content (sanitized)
card Rich content card with custom data

Accessibility Features

The chatbot includes robust accessibility options to ensure all users can interact effectively:

User Control Options

  • Font Size: Adjust text size from small to extra large
  • Reduced Motion: Minimize animations for users sensitive to movement
  • High Contrast: Enhance visual distinction between elements
  • Keyboard Shortcuts: Navigate the chatbot using keyboard commands
// Configure accessibility options
window.ChatbotWidget.init({
  // ...other options
  accessibility: {
    fontSize: "large",      // "small", "medium", "large", "xlarge"
    reducedMotion: true,    // true or false
    highContrast: false,    // true or false
    keyboardShortcuts: true // true or false
  }
});
Keyboard Navigation

Users can press Tab to navigate between controls, Enter to select options, Escape to close dialogs, and Alt+C to toggle the chatbot open/closed.

Implementation Note

For custom rich content types or specialized accessibility requirements, visit the Test Features page or contact our development team for integration assistance. All rich content features comply with WCAG 2.1 AA standards.

Frequently Asked Questions

Does the chatbot work on mobile?

Yes, our widget is fully responsive and adapts perfectly to screens of all sizes, from smartphones to large computer screens.

Can I modify the chatbot's appearance after installing it?

Absolutely! You can modify the configuration options at any time. If you're using the data-chatbot-config attribute, update the values and refresh the page. If you're using JavaScript initialization, call window.ChatbotWidget.init() again with your new options.

How can I further customize the chatbot's functionality?

For more advanced customizations, check our complete documentation on GitHub. There you'll find information about integrating with natural language processing APIs, customizing events, and much more.
